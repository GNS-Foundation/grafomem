Quick Start

# Install
pip install grafomem

# Write a memory
curl -X POST https://grafomem-production.up.railway.app/v1/stores/default/write \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"content": "User prefers dark mode", "options": {}}'

# Retrieve
curl -X POST https://grafomem-production.up.railway.app/v1/stores/default/retrieve \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"query": "What does the user prefer?", "options": {"top_k": 5}}'

API Documentation

          
Quickstart guide and endpoint reference


          
          

            
⚡ Quickstart

            
Get started with GRAFOMEM Cloud in 3 steps:

            

              
# 1. Get your API key from the API Key tab above

              
export GFM_KEY="gfm_..."

              

              
# 2. Create a memory store

              
curl -X POST https://cloud.grafomem.com/v1/stores \

              
  -H "Authorization: Bearer $GFM_KEY" \

              
  -H "Content-Type: application/json" \

              
  -d '{"name": "my-agent-memory"}'

              

              
# 3. Write your first fact

              
curl -X POST https://cloud.grafomem.com/v1/stores/{store_id}/write \

              
  -H "Authorization: Bearer $GFM_KEY" \

              
  -H "Content-Type: application/json" \

              
  -d '{"content": "The user prefers dark mode", "tags": ["preference"]}'
